STUDENT MANAGEMENT SYSTEM

About Project:
This is a Python-based Student Management System developed to manage student academic records efficiently.

The project allows users to:
- Add student details
- View all student records
- Search students
- Update marks
- Delete records
- Check grades and results
- Display class topper
- Show class statistics
- Save and load data using JSON

Technologies Used:
- Python
- JSON File Handling
- VS Code
- GitHub

Concepts Used:
- Functions
- Dictionaries
- Nested Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- File Handling
- Sorting & Ranking
